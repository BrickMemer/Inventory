# Inventory Simulator in c++
This project is about making inventory simulator inside of c++. Current version have a TUI design, and a store to buy and sell items.
## The concept of gameplay
Our concept of gameplay for this app is that player would buy new cheap items and sell them for a much higher. That's the main gameplay loop. In the future we plan to do saving current progress in json, and making a GUI interface.
## Current advantages of our program
  Lightweight.
  Fast performance.
  And small in size.
