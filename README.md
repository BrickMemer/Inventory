### Linux
To download the project you can download the zip from github, or clone the repository

#1 To download zip:
Click on button with the word "Code" on it, and press the button that says "Download ZIP", after unpack it and move to the next step

To clone the repository:
Click on button with the word "Code" on it, copy the link that shows on https tab, copy it and write in the terminal: 
```bash
git clone <the copied link>
```
To download it that way you need to have git

#2 To install first you need to execute this commands

```bash
cmake -S <the downloaded folder directory> -B <directory where do you want it to install into>
cmake --build <directory where did you tell it to install into>
```

#3 To run the app you need to open the terminal and copy the commands

```bash
cd <directory where did you tell it to install into>
./Inventory-New
```
